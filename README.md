#FTW-JS
> Libraries and components for FTW dApp clients.

## Included packages

- **@ftw-js/api**
- **@ftw-js/lottery-number-picker**
- **@ftw-js/fetch-paginate**
  
## Install

```sh
npm i @neo-react/[packages]
```

## Usage

```sh
npm run start
```

## Run tests

```sh
npm run test
```
